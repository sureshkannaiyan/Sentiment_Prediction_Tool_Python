Product Review Sentiment Predictor - Python

View on GitHub Download .zip Download .tar.gz

This tool predicts sentiment (opinions) from users comments (reviews) on a particular mobile phone product. Particularly, the sentiment on chosen project feature is predicted and results are shown in appropriate textboxes. The opinion of the users are classified to three categories: positive, neutral and negative. Machine learning algorithms (classification algorithms) of Python scikit-learn tool are used to classify user reviews.
